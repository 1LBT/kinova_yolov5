### 1. Launch文件
- 这将发布 m1n6s200_end_effector -> camera_link -> yolo_frame 的完整链条
- roslaunch ggcnn_kinova_grasping handeye_static_tf.launch
#### `handeye_static_tf.launch`
- 发布从末端执行器到相机的静态TF
- 包含最新的手眼标定参数
- 包含 `yolo_frame` 的定义，这对 `kinova_yolo_grasp.py` 至关重要
- **`yolo_frame`**: 专为 YOLO 脚本使用，修正了相机的旋转方向。
- 变换参数: `0 0 0 1.571 -1.571 0` (Yaw, Pitch, Roll)
- 父坐标系: `camera_link`
- **`camera_depth_optical_frame`**: 修正点云方向（Z向前）。
- 变换参数: `0 0 0 1.571 -1.571 0`
- 父坐标系: `camera_link`
### 2. 抓取程序
#### `kinova_yolo_grasp.py`
- 最新的主抓取程序，支持视觉伺服和抗奇异点策略
- 默认使用 `yolo_frame` 确保坐标变换正确
